### 🤖 AI 면접 준비

> AI와 함께 취업 준비의 핵심을 빠르게 완성해요.

**서류/공고 맞춤 분석**

- 이력서·포트폴리오와 채용공고를 함께 입력하면 강점, 부족한 점, 개선 방향을 분석  


**모의 면접**

- **인성 면접 모드** — 이력서·포트폴리오 기반 맞춤형 인성 질문
- **기술 면접 모드** — 직무 관련 기술 질문으로 실전처럼 연습
- 면접 종료 후 답변 자동 평가 및 개선 피드백 제공  

### 🗓️ 캘린더

> 캘린더로 취준 일정을 관리하고, 투두리스트로 할 일까지 함께 관리할 수 있어요.

- **월간 / 주간 뷰** — 전체 일정을 한눈에 파악
- **전형 단계별 색상 구분** — 면접, 서류, 코딩테스트를 색상으로 시각화
- **일정 목록** — 날짜별 지원 현황 한 번에 확인
- **투두리스트** — 오늘 할 일 등록 및 완료율 추적
- **구글 캘린더 연동** — 기존 일정과 자동 동기화

### ✍🏻 게시판

  > 이력서·포트폴리오·면접·코딩테스트 관련 경험과 정보를 공유하며 취준 기록을 함께 쌓아요.                         
                                                                                                                 
  - **정렬** — 최신순 / 인기순 / 팔로잉 피드 전환                                                                
  - **태그 필터** — 이력서, 포트폴리오, 면접, 코딩테스트 카테고리별 탐색
  - **댓글 · 좋아요** — 게시글에 반응하고 소통                                                                   
  - **개인정보 수동 마스킹** — 이미지, pdf 첨부 시 드래그로 영역을 직접 선택해 모자이크 처리                          
  - **파일 첨부** — 이력서·포트폴리오 파일 공유 가능

### 💬 1:1 채팅

  > 궁금한 점이 생기면 1:1 채팅으로 바로 질문하고 이력서·포트폴리오 피드백을 빠르게 주고받아요.                    
                                                                                                                 
  - **실시간 메시지** — WebSocket(SockJS + STOMP) 기반 즉시 전송                                                 
  - **PDF · 파일 공유** — 이력서·포트폴리오 파일을 채팅으로 바로 전달                                            
  - **팔로우** — 프로필에서 바로 팔로우하고 관심 사용자의 글을 피드에서 확인                                     
  - **안읽은 메시지** — 마지막 읽은 위치 기반 미확인 메시지 표시
